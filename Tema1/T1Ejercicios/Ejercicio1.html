Lenguajes de programación
1. ¿Qué diferencia existe entre los lenguajes declarativos y los imperativos? Nombra al menos 2 de cada tipo.

Los Declarativos indicamos el resultado a obtener sin especificar pasos (Lisp, SQL, Haskell)
Los Imperativos indicamos los pasos a seguir para obtener resultados (C, Java, C++)

 2. Explica qué es compilar. Explica qué es interpretar

Compilar: Todo el código fuente se traduce y se ejecuta
Interpretar: Se ejecuta el código fuente de línea en línea.

 3. Ventajas de los lenguajes compilados.

A la hora de ejecutar código es más eficiente que el interpretado

 4. Ventajas de los lenguajes interpretados.

Interpreta el código fuente directamente y es independiente de la máquina y del sistema operativo

 5. Nombra 2 lenguajes compilados y otros 2 interpretados.

Compilados serían C o C++
Interpretados serían PHP o Javascript
 6. ¿Puede considerarse código objeto el bytecode generado en Java tras la compilación? Explica la respuesta.

Si, porque el código objeto es un resultado de compilar código fuente
